# cm: Commit

Commit your changes

### Usage

```
$ gt cm [commit-message]
```

__Commit, letting git decide the editor for editing commit messages__

```
$ gt cm
```

which mimics

```
$ git commit
```

__Commit, adding the commit message in the command__

```
$ gt cm "Commit message here"
```

which mimics

```
$ git commit -m "Commit message here"
```
